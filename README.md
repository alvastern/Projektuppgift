# Projektuppgift - Frontend-baserad webbutveckling
I detta projekt har en webbapplikation för att visa väder skapats. En automatiserad arbetsprocess med Vite har använts tillsammans med versionshnatering med Git / Github. För styling har CSS-preprocessoren sass / scss använts.

Projektrapporten ligger i mappen i mitt repository för projektet eller hittar du den via länken nedanför.
